# 实战概念与先攻/后攻策略

## T0/T1/T2定义
- T0(先攻第一回合): 自己先攻, 第一回合做出终场
- T1(后攻第一回合): 对方先攻, 自己第一回合突破+反展开
- T2(后攻第二回合): 对方做出终场后, 自己第二回合才行动
- T0最优: 做出多阻抗终场, 对方难以突破
- T1可接受: 能突破对方场+反展开
- T2不理想: 对方已有准备, 且你的回合开始时对方手卡可能已有阻抗

## 起手率要求
- T1累计起手率≥80%: 可接受底线(用ydkcheck.py --section=start验证)
- T0起手率≥50%: 竞技级标准
- T2率<20%: 可接受底线
- 低于底线: 卡组卡手率过高, 需要优化启动轴

## 先攻(自己做场)
- 目标: 做出≥2阻抗终场
- 全展开: 从1-2张卡做出≥2阻抗终场
- 护航展开: 盖反击陷阱(灵王/自奏之阶)再启动, 保护combo不被断
- 理想终场类型:
  - 康+无效+清场(3-4阻抗): 自奏(丁吉尔苏+S:P+之阶)
  - 康+无效(2阻抗): 烙印(伽拉忒亚回收+丁吉尔苏)
  - 无效+站场(2阻抗): 深渊之兽(巴尔德+赫界龙)
- 阻抗来源: 额外怪兽康 + 场上永续 + 盖放陷阱 + 手坑预留

## 后攻(突破对方)
- 目标: 突破对方终场+反展开
- 解场卡优先级: 闪电风暴/黑洞/羽毛扫 > 指名者 > 指名卡
- 解场时机: 先清理自己场(闪电风暴)再用, 解场后保留至少1手坑防反击
- 多阻抗场: 先解最威胁的(康→无效→站场)
- 后攻突破资源: 解场魔法2-4 + 手坑9-12 + 泛用无效1-2

## 吃G妥协
- 触发条件: 起手或抽到增殖的G, 对方已发动G效果
- 原则: 特殊召唤越少越好, 减少对方抽卡收益
- 妥协展开: 通常召唤核心→做1无效, 盖反击/陷阱, 用墓地效果做阻抗
- 妥协场标准: ≥1康/无效 + ≥1站场怪兽
- 不要强行做 full combo, 对方抽卡收益可能直接获胜
- 可用妥协手段:
  - 通常召唤核心怪兽(不特招)
  - 盖1-2张反击/陷阱
  - 用墓地已有的资源做1个无效
  - 通召点留给有压制效果的怪兽

## 不同卡组的终场参考
- 自奏: 丁吉尔苏(康) + S:P小夜(康/除外) + 灵王之阶(反特招) = 3-4阻抗
- 烙印: 伽拉忒亚(回收+无效) + 丁吉尔苏(康) = 2阻抗
- 深渊之兽: 巴尔德(手卡特招+除外康) + 赫界龙(检索) = 2阻抗
- 肃声: 法理守护者(无效) + 虹光(康) = 2阻抗
- 黄金国: 黄金乡巫妖(墓苏生) + 永续陷阱连锁 = 1-2阻抗
- 单康妥协(1阻抗): 吃G后任何卡组的标准

## 解场优先级
- 1 priority: 康类效果(神之宣告/S:P/丁吉尔苏) — 最优先解决
- 2 priority: 无效类效果(技能抽取/独角兔) — 其次解决
- 3 priority: 站场怪兽(高攻/遗言) — 最后处理
- 解场后: 保留至少1手坑防对方反击, 不要全部交出

## 手坑使用时机
- 灰流丽: 对方检索/堆墓/特招效果发动时, 从手卡丢弃连锁
- 增殖G: 对方展开前发动(先攻盖放, 后攻直接发动)
- 墓穴指名者: 自己展开前发动, 护航combo不被灰/G断
- 无限泡影: 后攻突破时(自己场上无卡时可手卡发动), 或先攻护航(空场盖放)
- 效果遮蒙者: 自己主要阶段从手卡送墓, 无效对方怪兽效果(只能自己回合)
