# 手坑目录

- 定义: 卡片(怪兽或陷阱), 可在**对手回合**从**手卡**发动, 效果**干扰**对方行动
- 干扰=无效/破坏/除外/停止召唤/限制效果/阻止检索/对方召唤时抽卡(资源压制)
- 不满足以下任一条件≠手坑: (1)从手卡发动 (2)对方回合 (3)干扰对方
- 仅从手卡特招自己+无干扰副作用≠手坑(例: 电子龙/阿不思的落胤)
- 仅从手卡检索/展开+无干扰≠手坑(例: 阿莱斯特/雷电龙)
- 需场上有其他卡作为cost才能从手卡发动≠纯手坑, 归入nht(例: 恐龙摔跤手需恐龙在场)
- 推荐: 9-12张, ≥3种不同类型, 最低6张(灰3+G2+墓指1, 不推荐)
- 与gentrap.md可重叠: 若陷阱卡可从手卡在对方回合发动干扰, 同时出现在两文件
